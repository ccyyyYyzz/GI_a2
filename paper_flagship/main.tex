% ---- Paragraph 2: THE FROZEN BOUNDARY PARAGRAPH (R36 s2, four sentences,

Before asking whether the medium can become a second output, we tested whether
ordinary bucket imaging still had meaningful scene-recovery headroom. Across five
preregistered comparisons, changing the pattern family, redistributing
measurements, reordering the same patterns, replacing the reconstruction
algorithm, and using the full count statistics produced no material improvement
over the global operating rule; one change was strongly harmful. The design-side
searches and the inference-side tests therefore bracketed the same conclusion from
opposite directions: within the declared count-only experiment, the remaining
scene gain is below the threshold fixed in advance. This does not prove optimality
over every possible optical measurement, but it changes the next question from how
to squeeze another fraction of a decibel from the bucket count to what else the
hidden channel has already measured.
